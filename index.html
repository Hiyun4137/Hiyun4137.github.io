QQ飞车游戏小结

发表于 2020-03-30 | 分类于 手游 ， QQ飞车

QQ飞车手游简介

《QQ飞车手游》是深圳市腾讯计算机系统有限公司基于《QQ飞车》端游开发的首款3D赛车竞速休闲手机游戏，该作于2017年12月29日公开测试。
游戏讲述了疾风、流火、幻想3个飞车王国，崇尚赛车文化。三个国家的车手们，为了证明自己的实力，为了自己的国家荣誉，进行相互竞赛的故事。游戏拥有宠物、天赋等战斗培养元素，还引入边境战争、时空穿梭赛、48人淘汰赛、赛道之王等PVP玩法

为什么会写一个游戏的感想

在日常生活中，游戏其实已经成为了一部分。但是我觉得这样一直玩一个游戏而对此不作任何点评，我会相当的难受。可能是我有点强迫症！

游戏玩法

目前主流游戏玩法是经典竞速与经典道具。分为个人竞速/道具与组队竞速/道具，车辆+宠物贯穿飞车世界。社交玩法分为结婚系统+车队系统+师徒系统+休闲区系统。
其他的趣味玩法有拉力赛，分站赛，邂逅赛，追击赛，超能竞速赛，猪猪争夺战等等…

新手指南

  • 车辆篇
    车辆的话，推荐每一期的勋章车，相对于其他的点券车还有钻石车是相当的顶了。

  • 宠物篇
    宠物的话分为竞速宠与道具宠。重点讲宠物，因为宠物的耐久度比车辆高！
    竞速宠分为三个类别：

    1. 短图宠
    2. 长图宠
    3. 集气宠
      其中短图宠以申公豹与雪域狮王较强，长图宠以哪吒与雷震子较强，集气宠以小白龙与仙鹤老鹰较强。以上所说大部分为钻石宠。
      如果不想氪金也可以玩下去的。老鹰不就是点券宠吗，而且一定会返厂的。然后就是车队商店中的小羊了，有羊癫疯的称号喔，可见实力不可小觑。接着是点券宠物蛋中的未来机器人，虽然我没有用过，可它的技能我很看好。
      道具宠我不是主玩道具，不过也总结一下：
    4. 领跑宠
    5. 补位宠
    6. 辅助宠
      其实我不知道是不是这么分，可道具就是这三个位置。。就听我瞎扯一番吧！
      领跑顾名思义就是带队冲锋的，所以建议带个能给盾，或者能防控的宠物例如乌龟，自然之灵，美人鱼等
      补位是随时补领跑的位置，由于经常换位，进攻建议带一个给氮气或者能持续进攻的例如避水兽，白泽，彩虹马等
      辅助的话是三人中位置最为靠后的一位，虽热如此，但作用巨大。(一般mvp都是辅助哦)由于在后面，一般都是道具双吃来保证队伍的道具容错率。单体进攻道具给补位或者领跑来进攻敌方的前置位，而群体道具则是作为骚扰配合使用。所以宠物我认为稍微配置一个技能好一点道具宠或者不要都行。辅助看的是意识哦，例如墨汁，龙卷风，雷等等

  • 赛道篇
    如果你真的想好好的玩这游戏，那么建议你走我这一条路线(本人的游戏经验总结出来的)：

    1. 熟练掌握进阶出弯(双段)

    2. 熟练掌握甩尾

    3. 熟练掌握侧身

    4. 熟练掌握卡气

    5. 将双段进阶改为单段进阶

    6. 熟练掌握断氮

    7. 刚图(11城一张足以)

    8. 要求在刚11城的过程中熟练掌握单段进阶+双刹断氮+暴力入弯+暴力出弯

    9. 刚图(幻音城假日/故园归醉，顺序不分先后)

    10. 要求在刚幻音城能学习并且熟练掌握双刹弹射，故园归醉能了解氮气延续的基本知识。两者刚完能对路线调整有一个基本的认知

    11. 接下来就是你喜欢跑什么图就去了解相关图的跑法路线啦！所有的基本技巧都已经会了哦！是个车神大佬啦！

游戏经历

我是一名开服玩家，虽然S1赛季过后有一段日子没有碰过这游戏。不过后来遇见了我的室友，才慢慢的重新游戏飞车世界。
S1赛季实力星耀。不过当时的游戏玩法比较单一。再加上有着QQ飞车端游的游戏经验，还是对比端游有点相似度的。很容易上手，同时也很容易让人乏味(这是当时退游的原因之一)
因为跟着室友玩，再次接触的时候，转眼已经过去了八个赛季(不得不说，时间真的过得很快)。在S9赛季中，慢慢再次熟悉这款游戏。同时发现，游戏的玩法已经是大不一样。。进阶氮气出弯，侧身氮气出弯，侧身进阶氮气出弯，U甩，快出，暴力入弯，暴力出弯等等…
我发现我对这款游戏又充满了好奇，也开始学习新技巧。同时技术也慢慢上升，达到了星耀水平！在S10赛季的时候成功让室友带上车神！
接着，又是退游…
可能这就是我吧，心中达到一个阶段就不会再想着下一个阶段而是转移目标。我心里觉得我的这个游戏的目标已经达成，就没有了再玩下去的必要了。
转眼又是4个赛季，飞车世界迎来了S14赛季，同时也迎来了许久未归的我。。
在这个赛季，我对自己又有了一个全新的目标。那就是游戏的最高段位–>传奇车神。在这个赛季中，我学会了暴力出弯/暴力入弯/路线选择以及加深了对游戏的数据认知(基础动力，氮气动力，小喷动力等等)。在量变达到质变的情况下，我的十一城的记录也是达到了201。虽然可以进2分，我还是选择放弃。这两分钟可以进但是对我的现状并不是一件好事。男孩子总会选择带一点雀斑的小姑娘！
也是在这个赛季，我组了一个车队，小伙伴还是挺多的。收了两徒弟，想着把这两个小徒弟带上来。这样也算不孤单吧。。

初来hexo总结

发表于 2020-03-25

新人报道

看到这篇文章代表已经成功的按照各个大佬还有文档当中的说明，搭建好了自己的博客并且成功发表第一篇文章

总结

该个人博客采用的是hexo部署到github下的方式(到后面可能会转到其他平台，因为进github的速度太慢了毕竟是国外的。。)

  1. 先下载git，还有node.js。

  2. 注册github账号并且创建仓库。
    2.1 注册的仓库名一定是自己注册的github用户名[username.github.io]
    2.2 仓库要为公开，否则访问不了
    2.3 以上两点出错则网页会报错404(前提是你的配置文件无误)

  3. 注册ssh-key秘钥，将你的github账户连接到hexo上
    3.1 在git上输入命令[ssh-keygen -t rsa -C “注册github的邮箱”]
    3.2 找到生成的秘钥文件，(记住是pub结尾的这个是公钥，另外一个是私钥)将它链接到你的github账户上

  4. 下载hexo并且本地使用(可以进官网看文档)

  5. hexo发布至github(官网有操作步骤)
    5.1 hexo g 生成
    5.2 hexo d 部署
    5.3 hexo clean 清除缓存
    5.4 hexo s 本地预览

  6. 更换主题
